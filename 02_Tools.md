## Step 2: Tools
Once you have the materials, you will need some tools to work with those materials. Here is a list of tools used in this guide:

- Chop saw
- Skill saw
- Table saw
- Oscillating saw
- Air compressor
- Impact drill
- Tape measure
- Level
- Caulking gun
- Grinder
- Clamps

Consider getting at least these tools before starting your project.
